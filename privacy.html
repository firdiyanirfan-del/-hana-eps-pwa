Kebijakan Privasi

Terakhir diperbarui: Juni 2026

1. Data yang Dikumpulkan

EPS-TOPIK Hana mengumpulkan data berikut melalui login Google:

  • Nama dan email (dari akun Google Anda)
  • Foto profil (dari akun Google Anda)
  • Skor, XP, streak, dan riwayat kuis
  • Jawaban salah untuk fitur Smart Review
  • Pengaturan aplikasi (tema, bahasa, dll.)

2. Penggunaan Data

Data digunakan untuk: menyimpan progres belajar, menampilkan statistik, menyinkronkan data antar perangkat, serta meningkatkan pengalaman belajar. Data tidak dijual atau dibagikan ke pihak ketiga.

3. Penyimpanan Data

Data disimpan di server PostgreSQL (Neon) dengan enkripsi SSL. Data cadangan (cache) juga disimpan di browser (localStorage) untuk akses offline.

4. Keamanan

Semua koneksi menggunakan HTTPS. Autentikasi menggunakan Google OAuth 2.0 dan token JWT. Data kuis hanya bisa diakses oleh pemilik akun masing-masing.

5. Hak Anda

Anda dapat menghapus akun dan seluruh data kapan saja melalui halaman Profil (ikon profil pojok kanan atas → hapus akun). Penghapusan bersifat permanen.
